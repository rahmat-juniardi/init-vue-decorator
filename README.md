# How to Run Project

## Run Development Mode
1. Install dependencies by running
   ```
   npm install
   ```
2. Run dev command
   ```
   npm run dev
   ```

## How to Build
1. Install dependencies by running
   ```
   npm install
   ```
2. Run build command
   ```
   npm run build
   ```

## How to Production Mode
1. Install dependencies by running
   ```
   npm install
   ```
2. Run preview prod command
   ```
   npm run preview
   ```

## Development
This project is works well with npm `9.5.1` & node `20.12.2`
